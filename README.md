# Otimizando-a-Reten-o-de-uma-Newsletter-Digital
Este projeto foi desenvolvido com foco em inteligência de produto e estratégias de Growth para modelos de negócios baseados em assinaturas e mídia digital (como o *The News*). O objetivo principal não é apenas monitorar o volume de acessos, mas sim aplicar **raciocínio orientado a hipóteses** para mapear o comportamento de leitura
